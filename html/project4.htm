FTP Simulator



For this project, I developed a client-server net-centric program in C that uses the FTP and TCP protocols.

- The client program connects to the server side program using socket
- User must enter a valid FTP username and password
- User enters command and argument, command is sent to server
- Server program checks cmd and argument, performs appropriate implementation and returns message to client
- Client program prints reply message to user
- Example commands include: stat, help, ls, cd, pwd, rm, send, mkdir, rmdir, recv, quit
