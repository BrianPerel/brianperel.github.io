ATM Machine App



For this project, I developed a Java ATM simulation app.

- Account number must be 8 digits long. The pin must be 4 digits long. Enter S or C for Savings or Checkings account type
- ATM functions: select savings or checkings, balance inquiry, deposit, withdraw, transfer funds, terminate account, save and load account
- Save information to receipt file, prompt user if receipt is wanted, if not then receipt file is deleted
- Withdraw operation will be prevented if the balance is below the requested withdraw amount
- Deposit operation will be prevented if the deposit amount is extremely high
